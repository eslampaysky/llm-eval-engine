## [0.1.0] - 2026-03-13
### Added
- BreakerClient with break_model(), get_report(), list_reports(), delete_report()
- Typed Report, FailedTest, Metrics models
- GitHub Action for CI/CD integration
- Support for openai, huggingface, webhook target types
- FastAPI backend with API-key auth, rate limiting, usage tracking
- React/Vite dashboard with runs, targets, compare, playground pages
